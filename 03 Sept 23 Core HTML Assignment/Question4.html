1. <html>

    • Tagname: HTML Root Element
    • Use: It defines the root of an HTML document and contains all the other HTML elements on the page.

  2. <head>

    • Tagname: HTML Head Element
    • Use: It contains meta-information about the HTML document, such as the title, character set, and links to external resources like stylesheets and scripts.

  3. <title>

    • Tagname: HTML Title Element
    • Use: This tag sets the title of the HTML document, which is displayed in the browser's title bar or tab.

  4. <Meta>

    • Tagname: HTML Meta Element
    • Use: This tag sets the title of the HTML document, which is displayed in the browser's title bar or tab.

  5. <link>

    • Tagname: HTML Link Element
    • Use: It is used to link external resources, such as stylesheets (CSS), to the HTML document, allowing for consistent styling and layout.

  6. <script>

    • Tagname: HTML Script Element
    • Use: This tag is used to embed or reference JavaScript code within an HTML document to add interactivity and functionality to web pages.

  7. <body>

    • Tagname: HTML Body Element
    • Use: It defines the main content of the HTML document, including text, images, and other media. This is what users see in the browser.

  8. <body>

    • Tagname: HTML Division Element
    • Use: It is a generic container used to group and structure HTML elements. CSS is often applied to
      elements to control layout and formatting.

  9. <p>

    • Tagname: HTML Paragraph Element
    • Use: This tag defines a paragraph of text. It's commonly used for organizing and styling text content.

  10. <a>

    • Tagname: HTML Anchor Element
    • Use: This tag defines a paragraph of text. It's commonly used for organizing and styling text content.
